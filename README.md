# GAP
Genetic Analysis Program.


#### To convert 'SHAPEIT' output file ('.haps') to '.ped' file:
    gap --haps2ped --file [file_name] --out [file_out] --code01


# Download
    git clone https://github.com/rtahmasbi/GAP
    

# Compile
To compile GAP, run the following command:

    cd GAP
    make

# Help
    ./gap --help
